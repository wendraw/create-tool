# tool-kit

This is a front-end tool chain.

## Install

> **Compatibility Note:**
> Vite requires [Node.js](https://nodejs.org/en/) version >=12.0.0.

With NPM:

```bash
npm init tool
```

With yarn:

```bash
yarn create tool
```

With npx:

```bash
npx create-tool
```

With PNPM:

```bash
pnpx create-tool
```

Then follow the prompts!

---

Currently supported tool:

- `code lint`
  - use `eslint` and `prettier`
- `git hooks`
  - use `husky` and `commitlint`
